## ⚡ Getting Started

### Backend Setup

1. Navigate to the backend folder:

```bash
cd backend

npm install

Start the backend server:

npm run dev


Frontend Setup

Navigate to the frontend folder:

cd ../frontend

Install dependencies:

npm install

Start the frontend server:

npm run dev

http://localhost:5173/
